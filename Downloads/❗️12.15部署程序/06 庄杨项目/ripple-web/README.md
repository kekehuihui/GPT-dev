# 波纹 (Ripple) - 舆情分析平台

一个基于 React + Vite 构建的现代化股票舆情分析平台，提供实时市场情绪分析、个股深度研究和智能事件雷达功能。

## ✨ 功能特性

- 📊 **大盘概览**：实时指数监控、市场情绪仪表盘、板块热力图
- 🏆 **舆情龙虎榜**：全网声量排名、AI 情绪识别、核心驱动题材分析
- 🔗 **产业链图谱**：事件传导路径可视化、上下游关联分析
- 📈 **个股深度**：价格与舆情同步视窗、聪明钱 vs 傻瓜钱分析、社区讨论流
- 📉 **策略回测**：舆情因子策略回测、收益率分析、风险指标

## 🛠️ 技术栈

- **前端框架**：React 19
- **构建工具**：Vite 7
- **样式方案**：Tailwind CSS 4
- **图表库**：Recharts
- **图标库**：Lucide React

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:5173 查看应用

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录

### 预览生产构建

```bash
npm run preview
```

## 🌐 部署

本项目已配置好 Cloudflare Pages 部署。详细部署步骤请查看 [DEPLOY.md](./DEPLOY.md)

### 快速部署步骤

1. 将代码推送到 GitHub 仓库
2. 在 Cloudflare Pages 中连接 GitHub 仓库
3. 配置构建设置：
   - 构建命令：`npm run build`
   - 构建输出目录：`dist`
4. 点击部署，等待完成

### 数据接入计划

根据后端开发蓝图，需要接入以下数据源：

1. **基础行情数据**：使用 AkShare 或 Tushare Pro
2. **舆情文本数据**：爬取东方财富股吧、雪球、财联社
3. **资金流向数据**：Level-1/Level-2 数据接口
4. **事件日历数据**：财报日历、宏观事件、行业会议

详细的后端开发计划请参考项目根目录下的 `波纹 (Ripple)后端开发蓝图.txt`
